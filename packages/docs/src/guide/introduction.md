# 介绍

## 定义

`Vue Popup Plus` 是一个基于 `Vue3` 的弹出层插件，它提供了简单易用的 API，支持高度自定义的配置项，用于创建自定义的弹出层。

作为一款轻量级插件，无需依赖其他框架和组件库，只需要最基础的 `Vue` 环境即可使用。

## 优势

如果你喜欢 `高度模块化` 的设计，并且对于在页面内嵌入弹出层 `复杂` 而又 `难以维护` 的样式以及交互逻辑这一行为感到困扰，那么 Vue Popup Plus 绝对是您的理想选择。

它的主要 `优势` 包括：

### 函数式 API

它提供了`函数式 API` 的简洁使用方式，您可以在任何需要弹出层的地方直接调用函数进行渲染，而无需关注弹出层的创建和管理细节。

### 组件化设计

和目前主流的弹出层插件需要在父组件中通过模版插槽的渲染方式不同，所有弹出层渲染的内容均以 `组件引入` 的形式进行渲染，避免了在父组件中嵌入弹出层的复杂逻辑，做到了 `完全解耦` 和 `便于维护`。

### 高度可定制化

专注于弹出层的 `创建` 和 `管理` ，并且提供了 `丰富` 的渲染选项，您可以根据自己的需求进行定制。

### 强大的插件扩展能力

提供了强大的插件扩展能力，您可以轻松开发出独立的插件，便于独立维护和管理。

### 预置插件的高度解耦

基于其强大插件扩展能力，我们将常用功能作为 `预置插件` 提供，与核心功能高度解耦，提供更自由的 `选择` 和 `组合` 空间。
